# yt-dlp GUI — 使い方ガイド

YouTube をはじめとする様々なサイトの動画・音声を、  
かんたんな操作でダウンロードできるソフトです。

---

## リリースノート

### v1.1.0（最新）

**新機能**
- ダウンロード済みファイルをスキップする設定を追加。「オプション」の「ダウンロード済みをスキップ」をオンにすると、すでに同名ファイルが存在する場合はダウンロードをスキップします

## 動作環境

- **OS**: Windows 10 / 11（64bit）
- **インストール不要**: Python・FFmpeg など追加ソフトのインストールは必要ありません
- **ネット接続**: ダウンロード時にインターネット接続が必要です

## 起動方法

`ytdlp_gui.exe` をダブルクリックしてください。それだけです。

> ウイルス対策ソフトが警告を出す場合がありますが、誤検知です。  
> 「許可する」または「実行する」を選んでください。

## 基本的な使い方

**① URLを貼り付ける**

画面左上の入力欄にダウンロードしたい動画のURLを貼り付けます。  
複数の動画をまとめてダウンロードしたい場合は、1行に1URLで複数貼り付けられます。

URLは `Ctrl+V` で貼り付けるか、「クリップボードから貼り付け」ボタンでも追加できます。  
貼り付け前に重複・無効な URL がないか自動でチェックされます。

**② 保存先を選ぶ**

「保存先」欄の右側にある「参照…」ボタンから、保存したいフォルダを選んでください。  
前回使ったフォルダは自動で記憶されます。

**③ ダウンロード開始**

画面左下の「▶ ダウンロード開始」ボタンを押すと開始します。  
右側の画面にダウンロードの進捗・速度・残り時間が表示されます。

途中でやめたい場合は「■ 停止」ボタンを押してください。  
エラーが出たURLは「↺ 失敗を再試行」ボタンで再実行できます。

## 主な機能

### ダウンロードモード切り替え

「ダウンロードモード」で **動画** か **音声のみ** を選べます。

- **動画**: mp4 / mkv / webm などの形式、画質（最高品質〜360p）を選択可能
- **音声のみ**: mp3 / wav / flac / m4a などに変換して保存

### ファイル名の変更

「✏ ファイル名変更」欄に名前を入力すると、そのファイル名で保存できます。  
拡張子は自動で付与されます。

この機能は **単体動画のダウンロード時のみ** 有効です。  
プレイリストをダウンロードするときや、URLを複数入力したときは自動的に無効になります。

### ダウンロード済みをスキップ

「オプション」の「ダウンロード済みをスキップ」をオンにすると、同名ファイルがすでに存在する場合はダウンロードをスキップします。プレイリストを定期的にチェックして新着だけ保存したいときに便利です。

### 完了後アクション

「高度な設定」の「完了後アクション」で、全ダウンロード完了後の動作を設定できます。

- **何もしない**: そのまま待機します
- **フォルダを開く**: 保存先フォルダを自動で開きます
- **シャットダウン**: 確認ダイアログの後、30秒後にPCをシャットダウンします。大量ダウンロードを仕掛けて離席するときに便利です

### 設定プリセット

「💾 設定プリセット」セクションで、現在の設定を名前をつけて保存できます。  
「YouTube 動画用」「音楽用 mp3」「アーカイブ用 最高画質」など、用途ごとにプリセットを作っておくと便利です。

保存したプリセットはコンボボックスから選んで「読み込む」ボタンで呼び出せます。不要になったプリセットは「削除」ボタンで消せます。

### プレイリスト一括ダウンロード

「オプション」の「プレイリスト全体をDL」をオンにしてプレイリストの URL を貼り付けると、収録動画をまとめてダウンロードします。

### 字幕のダウンロード

「オプション」の「字幕をDL・埋め込み」または「自動生成字幕もDL」をオンにすると、動画ファイルに字幕を埋め込んで保存できます。

### 範囲を指定してダウンロード

「オプション」の「範囲ダウンロード」に開始・終了時間を入力すると、動画の一部だけを切り出してダウンロードできます。

例：`開始 00:01:30`　`終了 00:03:00`

### ログの書き出し

「ログ」タブの「💾 ログを保存」ボタンで、ログ欄の内容をテキストファイルに保存できます。  
エラーが出たときの記録や、ダウンロード結果の確認に使えます。

### 同じURLの再ダウンロード

一度ダウンロードが完了またはエラーになったURLは、再度URLを貼り付けて「ダウンロード開始」を押すことで再ダウンロードできます。  
現在ダウンロード中・待機中のURLは重複追加されません。

### ダークモード・ライトモード

画面右上のトグルスイッチで切り替えられます。次回起動時も設定が引き継がれます。

## うまくいかないときは

**ダウンロードが失敗する・エラーが出る**

「オプション」の「エラーを無視して続行」がオンになっているか確認してください。  
また、yt-dlp は頻繁に更新されるため、古いバージョンでは対応できなくなるサイトが増えることがあります。その場合は開発者にご連絡ください。

**指定した形式（mp3 など）に変換されない**

「ダウンロードモード」が **音声のみ** になっているか確認してください。  
動画モードのままだと、音声形式の指定は反映されません。

**メンバー限定・年齢制限の動画をダウンロードしたい**

ブラウザの Cookie ファイルを使ってログイン情報を渡すことができます。

1. ブラウザ拡張機能「[Get cookies.txt LOCALLY](https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc)」をインストール
2. 対象サイトにログインした状態で拡張機能を使い、`cookies.txt` を書き出す
3. 「高度な設定」の「Cookie ファイル」欄に、保存した `cookies.txt` を指定する

**YouTube でレート制限エラーが出る**

短時間に多くの動画をダウンロードしようとすると、YouTube 側から最大1時間のアクセス制限がかかることがあります。時間をおいてから再試行してください。  
「高度な設定」の「帯域制限」に `2M` などを設定すると再発を防ぎやすくなります。

**ウイルス対策ソフトが反応する**

PyInstaller（Python アプリのパッケージングツール）で作成した EXE ファイルは、一部のウイルス対策ソフトで誤検知されることがあります。ソフト自体に問題はありません。「許可」「ホワイトリストに追加」などの操作で回避してください。
